Blue Moon 1.1.0b30 - Linux light GTK recovery test

Derived from: Blue Ice Reloaded 1.4.14062013
Target platform: Pale Moon 29.3.0 - 34.*

Changes since 1.1.0b29:
- restricted hard dark Add-ons Manager bitmap/palette rules to the dark-theme media path
- fixed the issue where the dark `ice-dark.png` Add-ons Manager background could remain active after returning to a light GTK theme
- kept system-color text and generic XUL add-on window rules adaptive for both light and dark GTK
- removed global tree row border forcing that caused visible row separators / extra-looking spacing in the Bookmarks sidebar
- added Phase E light GTK recovery notes
- updated release metadata to version 1.1.0b30

Blue Moon 1.1.0b31 - Linux adaptive Add-ons Manager background test

Changes since 1.1.0b30:
- replaced the `@media (-moz-dark-theme)` Add-ons Manager switch with an adaptive system-color blend test
- keeps `ice.png` as the Add-ons Manager identity image and blends it with `-moz-Dialog`
- avoids relying on `ui.color_scheme` / `-moz-dark-theme`, which does not automatically track GTK theme changes in Pale Moon/UXP
- preserves the b30 sidebar/tree row-spacing fix
- updated release metadata to version 1.1.0b31

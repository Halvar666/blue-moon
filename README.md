Blue Moon

This package is an unofficial Pale Moon port/adaptation based on Blue Ice Reloaded by IceCream.

Blue Ice Reloaded is itself derived from the earlier Blue Ice theme by Murry.

Original theme lineage:
- Blue Ice by Murry
- Blue Ice Reloaded by IceCream
- Blue Moon Pale Moon port/adaptation by Halvar

Additional credits:
- Contributor: Jivko Evgeniev

This port preserves and adapts that theme lineage for current Pale Moon compatibility.
